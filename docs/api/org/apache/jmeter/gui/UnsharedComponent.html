Interface UnsharedComponent

  • All Known Implementing Classes:
    AbstractVisualizer, AssertionVisualizer, BeanShellListener, ComparisonVisualizer, GraphVisualizer, HttpMirrorControlGui, MailerVisualizer, MonitorHealthVisualizer, PropertyControlGui, ProxyControlGui, RespTimeGraphVisualizer, SimpleDataWriter, StatGraphVisualizer, StatVisualizer, SummaryReport, TableVisualizer, ViewResultsFullVisualizer

    public interface UnsharedComponent
    Marker interface indicating that an instance of a component cannot be shared. The GUI instance will be shared among all test elements of a given type if the GUI component class does not implement this interface.
